Загрузка карт

Импорт данных о лицензиях из файлов GeoJSON

← Вернуться к карте

Инструкция по загрузке

  • Выберите GeoJSON файл с картой лицензий
  • Файл должен содержать полигоны с информацией о лицензиях
  • Поддерживаются форматы: .geojson и .json
  • Все полигоны автоматически добавятся на единую карту
  • Если лицензия уже существует, данные будут обновлены

Для пользователей Конструктора Яндекс.Карт:
После создания полигона на карте, в поле "Описание" впишите данные о лицензии в указанном ниже формате. Каждую строку разделяйте нажатием Enter (в GeoJSON это будет <br/>).

Полное описание лицензии

Образец текста для поля "Описание" полигона:

ХАБ 12345 НЭ Северный участок Площадь: 150.5 кв.км ООО Золотодобыча Дата выдачи: 15.03.2020 Действует до: 31.12.2030 Полезное ископаемое: Золото

Система автоматически заполнит поля:

Номер лицензии: ХАБ 12345 НЭ
Вид пользования: НЭ
Регион: Хабаровский край (автоматически по коду "ХАБ")
Участок: Северный участок
Площадь: 150.5 кв.км
Недропользователь: ООО Золотодобыча
Дата выдачи: 15.03.2020
Дата окончания: 31.12.2030
Полезное ископаемое: Золото

Минимальное описание

Если некоторых данных нет, просто пропустите эти строки:

ЮСХ 98765 БР ООО Сахалин-Ресурс

Заполнятся только указанные данные:

Номер: ЮСХ 98765 БР
Регион: Сахалинская область
Недропользователь: ООО Сахалин-Ресурс
Дата выдачи: Сегодняшняя дата
Ископаемое: Не указано

Распознаваемые ключевые слова

Поле Формат
Номер лицензии ХАБ 12345 НЭ (с пробелами между частями)
Площадь Площадь: 150.5 кв.км
Недропользователь ООО Название (укажите форму: ООО, АО, ПАО)
Дата выдачи Дата выдачи: 01.01.2020 или Выдана: 01.01.2020
Дата окончания Действует до: 31.12.2030 или Дата окончания: 31.12.2030
Ископаемое Полезное ископаемое: Золото или Ископаемое: Каменный уголь
